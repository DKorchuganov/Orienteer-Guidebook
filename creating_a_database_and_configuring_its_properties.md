# Creating a Database and Configuring its Properties

How to create a new database? How to set its URL?

You can configure the database on the **Schema** page (URL */schema*), on the **Database** tab. Parameters:
* **Type**
* **Status**
* **Default Cluster Id**
* **Date Format**
* **Date/Time Format**
* **Timezone**
* **Locate Country**
* **Locale Language**
* **Charset**
* **Custom**
* **Cluster Selection**
* **Minimum Clusters**
* **Conflict Strategy**
* **Validation***
